# DevRamp AI - Live Demo Script

**Total Time:** ~4.5 minutes

## 1. Introduction (0:30)
- "Welcome to DevRamp AI, the intelligent developer onboarding platform powered by IBM Bob."
- Show the landing page/dashboard.

## 2. Connect Repository (1:00)
- Click "Add Repository".
- Paste the URL of a sample public repo.
- Show the loading state: "IBM Bob is analyzing the entire codebase structure..."

## 3. Review Architecture (1:00)
- Once complete, open the **Explorer** view.
- Show the visual dependency graph generated by the backend and Bob service.
- "Bob instantly identifies the architecture, giving a new developer an immediate bird's-eye view."

## 4. AI Mentor Chat (1:00)
- Open the **Mentor** chat interface.
- Ask: "Where is the authentication logic handled in this repo?"
- Show Bob's natural language response highlighting the exact file and lines.
- Ask: "Can you explain how the JWT is verified?"
- Show the code explanation.

## 5. Learning Path & Wrap-up (1:00)
- Show the **Learning** tab.
- "Based on the repository complexity and the user's BEGINNER level, Bob generated this custom onboarding curriculum."
- Show the progress tracker.
- "DevRamp AI reduces onboarding from weeks to days."
